# 风云榜 (ScoreTrack)

一个轻量级的游戏积分管理应用，专为线下游戏场景设计。

## 功能特性

### 核心功能
- **活动管理**：创建、进行、结束活动全流程
- **玩家管理**：支持 2-10 人，自动分配颜色标识
- **积分录入**：灵活的积分单位（1/2/5/10/50/100），快速加减分
- **实时排行**：自动计算排名，实时更新排行榜
- **轮次历史**：查看、编辑、删除历史轮次
- **数据持久化**：localStorage 本地存储，无需联网

### 用户体验
- **响应式设计**：适配手机和桌面
- **主题切换**：light / dark / auto 三种模式，跟随系统偏好
- **路由恢复**：刷新页面保持当前位置
- **分享功能**：生成活动结果图片
- **扑克语录**：首页随机展示精选扑克智慧语录（中英文对照）

## 技术栈

- **前端**：Vanilla JavaScript (无框架依赖)
- **样式**：Tailwind CSS (构建输出，非 CDN)
- **构建**：Tailwind CLI
- **存储**：localStorage
- **图标**：Material Symbols
- **字体**：Lexend

## 快速开始

### 开发环境
```bash
# 进入项目目录
cd game-scoreboard

# 安装依赖
npm install

# 开发模式（监听 CSS 变化）
npm run watch

# 启动静态服务器（另一个终端）
npm run dev

# 访问 http://localhost:3000
```

### 生产构建
```bash
# 构建 CSS
npm run build

# 部署 src/ 目录到服务器
```

### 自动部署
推送到 main 分支会自动触发 GitHub Actions 部署。

## 使用说明

1. **创建活动**：填写活动名称、地点（可选）、玩家昵称、积分单位
2. **录入积分**：点击"录入本轮"，为每个玩家加减分，保存后自动更新排名
3. **查看历史**：点击"查看全部"查看所有轮次，可编辑或删除
4. **结束活动**：点击"结束活动"，生成最终排名和分享图片
5. **主题切换**：右上角按钮切换 light → dark → auto
